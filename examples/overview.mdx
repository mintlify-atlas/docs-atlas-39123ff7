## What You'll Learn

These examples cover the essential patterns for working with Phenomenon:

- **Attributes**: Define per-particle data like position, color, and custom properties
- **Uniforms**: Share values across all particles for synchronized animations
- **Shaders**: Write GLSL code to position and color particles on the GPU
- **Performance**: Leverage multipliers and GPU rendering for millions of particles

## Before You Start

Make sure you have Phenomenon installed:

```bash
npm install phenomenon
```

All examples assume you have a canvas element in your HTML:

```html
<canvas id="canvas"></canvas>
```

<Tip>
  Each example includes complete, runnable code with detailed explanations of how the shaders and attributes work together.
</Tip>
